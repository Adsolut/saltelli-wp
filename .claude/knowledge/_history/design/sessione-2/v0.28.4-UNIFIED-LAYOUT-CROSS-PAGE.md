## Issue (segnalata Duccio)

> "Falo per tutte le pagine"

Dopo v0.28.3 (footer pre-CTA + newsletter 3fr/9fr) il footer matchava le pagine editoriali (chi-siamo, tier-1) ma NON la home (single-col). User ha richiesto unificazione TOTALE cross-page.

## Fix v0.28.4 — Global section-head 3fr/9fr

```css
@media (min-width: 1024px) {
    .sl-section-head {
        grid-template-columns: 3fr 9fr !important;
        gap: 64px !important;
        align-items: start;
    }
}
```

Override globale del default `.sl-section-head { grid-template-columns: 1fr }`.

Markup compatibile esistente (zero PHP changes):
```html
<div class="sl-section-head">
    <div class="sl-mono">§ 01 — Aree di pratica</div>     ← grid col 1 (3fr) x=72
    <h2 class="sl-section-title">Diciannove aree...</h2>   ← grid col 2 (9fr) x=444
</div>
```
