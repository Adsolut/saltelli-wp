## Fix v0.27.3 — Revert ai valori JSX

```css
/* H1: scale-up rimosso, restituito JSX exact */
.sl-chi-siamo__h1 {
    font-size: clamp(56px, 7vw, 104px) !important;  /* era 9vw,144 */
    line-height: 0.98;                               /* era 0.92 */
    max-width: none;                                 /* era 12ch */
}

/* Founding § 02: 3-col asym restituito */
.sl-chi-siamo__founding-grid {
    grid-template-columns: 3fr 7fr 2fr;              /* era 3fr 9fr */
}

/* Team portrait: aspect 3/4 restituito */
.sl-chi-siamo__team-portrait {
    aspect-ratio: 3 / 4 !important;                  /* era 4 / 5 */
}

/* Principles § 04: STACKED rows restituite */
.sl-chi-siamo__principles-grid {
    grid-template-columns: 3fr 9fr !important;       /* era 1fr */
}
.sl-chi-siamo__principles-list {
    grid-template-columns: 1fr !important;           /* era repeat(3,1fr) */
}
.sl-chi-siamo__principle {
    grid-template-columns: 80px 1fr !important;     /* num + body */
    border-bottom: 1px solid var(--border) !important;
    border-top: 0 !important;
    background: transparent !important;
}
.sl-chi-siamo__principle-t {
    font-size: 32px !important;                      /* era 26px */
}
```

## Lesson learned

**v0.25.0 + v0.27.0 contraddizione prompt vs JSX:** quando il prompt utente chiede una direzione che diverge dalla fonte JSX approvata, JSX deve prevalere come ground truth. Le decisioni "extra" (h1 scale-up, principi 3-col, team 4:5, founding 3/9) erano deviazioni non-JSX-faithful — sono state revertite.
