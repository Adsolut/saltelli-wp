## Strategy decision T1

L'audit v0.25.0 segnalava `Person schema = 0` per le 4 page lawyer. Verifica diretta della response HTML:
```
"@type":["Person","Attorney"]
"alumniOf":{"@type":"CollegeOrUniversity","name":"Università degli Studi di Napoli Federico II"}
"honorificPrefix":"Avv."
```
Schema **già emesso** da `inc/schema/partial-attorney.php` (linea 143, `saltelli_emit_jsonld($person)`). L'audit grep falliva perché `@type` è array `["Person","Attorney"]` non string `"Person"`.

**Decisione:** NON aggiungere Yoast filter — creerebbe duplicate Person nel grafo, con conseguente warning Schema validator e potenziale demote da knowledge graph.
