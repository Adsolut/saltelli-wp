## Pattern handoff documented in CLAUDE.md

Aggiunta golden rule "Design → Code handoff":
1. Code DEVE mappare ogni inline style a className BEM
2. Code DEVE generare CSS rule corrispondente
3. Code NON deve assumere className mancanti = optional
4. Test pixel-perfect: 1 inline = 1 CSS + 1 className
5. Verifica computed-styles via Playwright post-deploy
