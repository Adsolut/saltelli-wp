## Note tecniche walkthrough

### Reveal animation artifact (Playwright headless)
Sezioni con GSAP/data-reveal mostravano `opacity:0` nei primi screenshot perché:
- IntersectionObserver non triggera reliably in fullPage capture
- GSAP ScrollTrigger non firing senza scroll utente

**Fix script v0.27.2:** force-reveal universale in evaluate():
```js
document.querySelectorAll('*').forEach(el => {
    if (el.style && el.style.opacity === '0') {
        el.style.opacity = '1'; el.style.transform = 'none'; el.style.translate = 'none';
    }
});
```
Strippa GSAP inline opacity:0 + transforms su TUTTI gli elementi pre-screenshot.

**Per utenti reali su Chrome reale:** scroll-driven reveal funziona normalmente. NON è un bug live.

### Stripping inline styles
Lo strip include:
- GSAP `style.opacity === '0'` → 1 + transform/translate none
- `[data-reveal]` + `.sl-revealable` → add `.is-revealed` class
- `.sl-word` (split title animation) → opacity 1 + transform none
