  <link rel="preload" as="style" id="..." href="..." onload="this.onload=null;this.rel='stylesheet'">

## Issue residui per ulteriore push (v0.22 candidate)

### Critical per ≥92 Performance mobile

1. **Reduce unused CSS** -29 KiB cross-page: sections.css contiene tutto, ogni template usa ~30%. **Solution**: per-template CSS chunk via conditional `wp_enqueue_style` in `inc/enqueue.php` (es. `is_singular('competenza')` → load competenza-only chunk). Stima Perf +5/-8pt.
2. **/contatti/ outlier (Perf 68)**: CF7 plugin carica proprio CSS/JS pesante. Render-blocking 900ms residuo. **Solution**: dequeue CF7 frontend assets ovunque salvo `/contatti/` via `wp_dequeue_style/script` su pagine non-form. Stima Perf +15-20pt su altre URL, /contatti/ stays baseline.
3. **Reduce unused JavaScript** -20-21 KiB: GSAP carica su tutte le pagine; alcune non animano nulla. **Solution**: GSAP enqueue conditional su `is_front_page()` o `body.has-animations`. Stima Perf +3-5pt.
4. **Forced reflow** flagged on /: GSAP ScrollTrigger con `is-revealed` class touch causa reflow. Indagare main.js GSAP setup. Stima TBT -50ms.

### Critical per A11y ≥95 cross-page

5. **Background contrast** ancora flagged: probabile `--text-muted` su `--surface` darker non scoped, OR navy text su qualcosa borderline. Run `axe-core` o Lighthouse details audit per identificare elementi.
6. **Accessible names mismatch** flagged: link/button con `aria-label` divergente da testo visibile. Audit pattern `<button aria-label="X"><span>Y</span></button>` cross-template.
7. **Touch targets** persiste su / mobile — il fix è scoped al footer, alcuni elementi homepage (sl-areas filter chips?) restano <48px.

### Critical per SEO ≥95

8. **Page is blocked from indexing** = staging robots noindex INTENZIONALE. Su produzione cut, removed → +30/-40pt. **No fix needed pre-cut**.
9. **Document does not have a meta description**: Yoast genera fallback ma non sempre. Audit `inc/seo/meta-tags.php` per ensure description present su tutte le pagine. Stima SEO +5pt.
