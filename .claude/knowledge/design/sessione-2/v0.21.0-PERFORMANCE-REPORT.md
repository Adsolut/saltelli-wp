### Task 7 — Cache headers + nginx hardening ⏸ MANUAL APPLY
**Stato corrente sul droplet (saltelli-staging-ams3-01):**
- ❌ NO Brotli module (`nginx -V` non lo lista)
- ❌ NO custom cache-control rules in `sites-enabled/*.conf`
- ❌ NO security headers (`X-Frame-Options`, `Referrer-Policy`, ecc.)
- ✅ nginx 1.24 valido, gzip presumibly default-on

**Recommended nginx snippet** (apply manualmente con `sudo nano /etc/nginx/sites-available/saltelli` + `sudo nginx -t` + `sudo systemctl reload nginx`):

```nginx
# === Cache headers === (immutable per asset versioned, breve TTL HTML)
location ~* \.(woff2|woff|avif|webp)$ {
    expires 1y;
    add_header Cache-Control "public, immutable, max-age=31536000";
    access_log off;
}
location ~* \.(css|js)$ {
    expires 1y;
    add_header Cache-Control "public, immutable, max-age=31536000";
}
location ~* \.(jpg|jpeg|png|gif|ico|svg)$ {
    expires 6M;
    add_header Cache-Control "public, max-age=15552000";
}
location ~* \.(html|php)$ {
    expires 5m;
    add_header Cache-Control "public, max-age=300, must-revalidate";
}

# === Security headers ===
add_header X-Content-Type-Options "nosniff" always;
add_header X-Frame-Options "SAMEORIGIN" always;
add_header Referrer-Policy "strict-origin-when-cross-origin" always;
add_header Permissions-Policy "geolocation=(), microphone=(), camera=()" always;

# === Brotli (richiede module install) ===
# sudo apt install nginx-module-brotli
# Aggiungi in nginx.conf top-level:
# load_module modules/ngx_http_brotli_filter_module.so;
# load_module modules/ngx_http_brotli_static_module.so;
# Poi nel server block:
brotli on;
brotli_comp_level 6;
brotli_types text/css application/javascript application/json text/html font/woff2 image/svg+xml;
brotli_static on;
```

**Risk note:** modifiche nginx richiedono `sudo nginx -t` PRIMA di `systemctl reload`. Errori syntax bloccano il server. Non eseguito senza supervisione esplicita.
