## Lesson learned

**`.sl-section-title { margin: 0 }` global rule** è una landmine cascade:
- Specificità 0,1,0 (uguale a `.sl-chi-siamo__h2`)
- Source-order indeterminato in alcuni casi (cascade engine non sempre garantisce later-wins)
- Soluzione: selettori scoped `.parent h2.section-class` (specificità 0,2,1) + `!important`

JSX usa **gerarchia tipografica differente per H2**:
- Body sezioni piccoli: clamp(28-32, ?, 36-44) — section subhead
- Body sezioni medi: clamp(32-40, ?, 48-64) — section heads
- CTA finali huge: clamp(40-56, ?, 64-96) — closing huge

**Strategia futura:** evitare global `.sl-section-title { margin: 0 }`. Usare scope per sezione/template per evitare cascade collisions.
